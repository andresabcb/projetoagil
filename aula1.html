Introdução à Engenharia de Software

Engenharia de Software é a disciplina responsável por desenvolver sistemas computacionais de maneira eficiente, confiável e escalável.

Processos de Desenvolvimento de Software

Engenharia de Software não se resume a programação; ela inclui planejamento, qualidade, segurança e manutenção.

Atividades do Processo de Software

  • Especificação dos requisitos
  • Projeto e implementação
  • Validação
  • Evolução
